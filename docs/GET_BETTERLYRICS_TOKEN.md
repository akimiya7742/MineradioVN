# Get BetterLyrics token
> [!NOTE]
> The BetterLyrics token is only valid for 12 hours - 1 day.
## Steps
- Download and install these extensions
    - BetterLyrics for Youtube Music: [Chrome](https://chromewebstore.google.com/detail/effdbpeggelllpfkjppbokhmmiinhlmg?utm_source=item-share-cb)
    - BTLGetToken: [Chrome](https://drive.google.com/file/d/1WPM8E3BjPZtvkZFjQ8MpG0zkrPb0bVcF/view?usp=sharing) | [See how to install custom extension](#install-custom-extension)
- Visit [Youtube Music](https://music.youtube.com) in your extension-installed browser
- Search for any tracks you'd like (make sure to select track / music, not music video)
- Navigate to Lyrics tab
- Open the BTLGetToken extension on the sidebar (depending on your browser)
- Copy the `token` field in the Payload section, this is your BetterLyrics token.

## Install custom extension
- Extract extension `.zip` file and save it wherever you want
- Head to extension settings (Settings->Extension->Manage extensions)
- Toggle Developer mode on
- Click load unpacked and load the extracted extension
